# agentspaces

Exploring the state layer for AI agents — durable, branchable workspaces that outlive any single compute session.

## Background

This project builds on research into Archil, Mesa, and Code Storage — three companies converging on the same bet: compute for agents is commoditizing, and durable state is the layer that matters. The full report, including 50 agent use cases, a comparison of six infrastructure paradigms, and the proposed "branchable agent workspace" primitive, lives in [docs/agent-infra-research-archil-mesa-code.md](docs/agent-infra-research-archil-mesa-code.md).

> State is the noun. Compute is a verb. Branches are control flow. Commits are the audit log.
